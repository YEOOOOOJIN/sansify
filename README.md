# Comic Sansify

> The world deserves this.

A Chrome extension that transforms all text on any webpage into Comic Sans MS.

## Install (Chromium-based browsers)

1. Download or clone this repo
2. Open your browser and go to `chrome://extensions/`
3. Enable **Developer mode** (top right toggle)
4. Click **Load unpacked** (top left)
5. Select this project folder

The extension icon will appear in your toolbar. Click it to toggle Comic Sans on/off for the current tab.

## Supported Browsers

- Chrome
- Edge
- Brave
- Opera
- Vivaldi

## How It Works

- Click the extension icon and use the toggle to enable/disable Comic Sans on the current tab
- When enabled, all text on the page renders in Comic Sans MS
- State is remembered per tab
